# Todo App - Frontend


App de notas fullstack con autenticación JWT. Cada usuario puede crear, editar y eliminar sus propias notas.


## 🛠️ Tech Stack


- React + Vite
- React Router DOM
- Axios
- Tailwind CSS + DaisyUI
- React Toastify


## ✨ Features


- Registro e inicio de sesión con JWT
- Rutas protegidas — sin login no entrás
- CRUD completo de notas
- Cada usuario ve solo sus propias notas

## ⚙️ Correr localmente


1. Clonar el repositorio
2. Instalar dependencias
\```
npm install
\```
3. Crear archivo `.env` con:
\```
VITE_APP_URL=http://localhost:3000
\```
4. Correr el proyecto
\```
npm run dev
\```
